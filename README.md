## Installation
Add this line to your application's Gemfile:

```ruby
gem 'breadcrumbs_on_rails-json_ld'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install breadcrumbs_on_rails-json_ld
```
